# netlivue

Vue 3 application with TypeScript and Tailwind CSS for Spotifun frontend.
Built with Vite.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### More information
See [vite][vite].
